2.1 EXISTING SYSTEM
--------------------

Currently, most educational institutions and organizations use the
following methods for attendance and monitoring:

  a) MANUAL ATTENDANCE SYSTEM:
     - Teachers/professors call out names from a register
     - Students sign on paper-based attendance sheets
     - Data is later manually compiled into spreadsheets
     - Limitations: Slow, proxy-prone, error-prone, no analytics

  b) BIOMETRIC ATTENDANCE (Fingerprint/RFID):
     - Uses fingerprint scanners or RFID cards
     - Requires physical contact or carrying a card
     - Limitations: Slow for large groups, hygiene issues, card loss,
       high hardware cost, maintenance required

  c) TRADITIONAL CCTV SURVEILLANCE:
     - Cameras record video continuously
     - Security personnel monitor feeds manually
     - Limitations: Requires human attention, no automatic identification,
       footage reviewed only after incidents

  DRAWBACKS OF EXISTING SYSTEMS:
  - No real-time identification capability
  - High dependency on human operators
  - Prone to errors and manipulation
  - Expensive hardware and maintenance
  - No integration between attendance and security


2.2 PROPOSED SYSTEM
--------------------

The proposed "AI Face Recognition and Monitoring System" overcomes the
limitations of existing systems by providing:

  a) CONTACTLESS IDENTIFICATION:
     - Uses camera-based face recognition (no physical contact needed)
     - Hygienic and user-friendly

  b) REAL-TIME PROCESSING:
     - Faces are detected and recognized in real-time from live video feed
     - Attendance is marked instantly upon recognition
     - No queues or waiting time

  c) AUTOMATED ATTENDANCE:
     - System automatically logs attendance with timestamp
     - Eliminates proxy attendance (each face is unique)
     - Records stored in database for easy retrieval

  d) SECURITY MONITORING:
     - Unknown/unregistered faces are flagged automatically
     - Alert notifications for potential intruders
     - All detected faces are logged for audit trail

  e) WEB-BASED DASHBOARD:
     - User-friendly interface accessible via web browser
     - View live feed, manage registered users, generate reports
     - No special software installation needed for end users

  f) COST-EFFECTIVE:
     - Runs on standard laptops/desktops with a webcam
     - Uses free, open-source software and libraries
     - Minimal hardware investment required

  ADVANTAGES OF PROPOSED SYSTEM:
  - High accuracy face recognition (99.38% on LFW benchmark)
  - Fully automated process
  - Contactless and hygienic
  - Real-time processing
  - Easy to deploy and maintain
  - Scalable architecture
  - Dual-purpose: attendance + security

2.4 LITERATURE SURVEY
----------------------

  1. FACE DETECTION USING HAAR CASCADE CLASSIFIER:
     Viola and Jones (2001) proposed a rapid face detection method using
     Haar-like features and cascade classifiers. This method is fast and
     works well for frontal face detection but struggles with varying
     angles and lighting conditions.

  2. HISTOGRAM OF ORIENTED GRADIENTS (HOG):
     Dalal and Triggs (2005) introduced HOG descriptors for human
     detection. This method captures edge and gradient structures and is
     more robust than Haar cascades for face detection under varying
     conditions.

  3. DEEP LEARNING-BASED FACE RECOGNITION:
     Schroff et al. (2015) from Google developed FaceNet, which uses a
     deep convolutional neural network to learn a mapping from face images
     to a 128-dimensional Euclidean space where distances correspond to
     face similarity. This approach achieved 99.63% accuracy on the
     Labeled Faces in the Wild (LFW) dataset.

  4. dlib's FACE RECOGNITION MODEL:
     King (2009) developed dlib, a toolkit containing machine learning
     algorithms including a ResNet-based face recognition model that
     achieves 99.38% accuracy on the LFW benchmark. The face_recognition
     Python library wraps dlib's model for easy use.

  5. OPENCV (Open Source Computer Vision Library):
     Bradski (2000) created OpenCV, which has become the most widely used
     computer vision library. It provides tools for image processing,
     video capture, face detection, and more. It supports Python, C++,
     and Java.

  6. RELATED PROJECTS:
     - Smart Attendance System using Face Recognition (IEEE 2019): Used
       a Raspberry Pi with camera module and OpenCV for classroom
       attendance.
     - Real-Time Face Recognition Attendance System (IJERT 2020):
       Implemented using Python and compared PCA, LDA, and deep learning
       approaches for recognition accuracy.
     - Face Recognition based Surveillance System (IJCST 2021):
       Combined face recognition with motion detection for enhanced
       security monitoring.
