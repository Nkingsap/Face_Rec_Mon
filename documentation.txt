                         ABSTRACT
================================================================================

The "AI Face Recognition and Monitoring System" is a real-time,
intelligent surveillance and attendance management solution developed using
Python and modern machine learning libraries. In today's world, security
and efficient monitoring are critical needs for educational institutions,
corporate offices, and public spaces. Traditional manual methods of
attendance tracking and security monitoring are time-consuming, error-prone,
and inefficient.

This project addresses these challenges by implementing a face recognition
system powered by computer vision and deep learning techniques. The system
uses OpenCV for real-time video capture and image processing, the
face_recognition library (built on dlib's state-of-the-art deep learning
model) for accurate face detection and identification, and a Flask-based
web interface for user interaction and monitoring.

A key highlight of the system is its robust two-step face registration
workflow: the administrator captures 50 photos of a new user directly
through the browser's live camera feed (using the browser-native
getUserMedia API), and those photos are saved to disk without immediately
impacting recognition. The admin can then review all unregistered users in
a dedicated "Train Model" panel and trigger model training with a single
click. Training processes all saved photos, computes an averaged
128-dimensional face encoding for each user, and updates the recognition
model — resulting in significantly higher accuracy than single-photo
registration.

Key features of the system include:
  - Real-time face detection and recognition from live camera feeds
  - Browser-native live camera preview during face registration
    (no server-side camera conflict; works over localhost/HTTPS)
  - Multi-photo capture: 50 frames captured automatically at 350ms
    intervals with a real-time progress bar and thumbnail preview strip
  - Two-step training workflow: capture photos first, train the model
    separately via the dedicated Train Model panel
  - Untrained user list showing all users pending model training
  - Averaged face encodings from multiple samples for improved accuracy
  - Automated attendance logging with timestamps
  - Web-based dashboard for monitoring and report generation
  - Attendance history and analytics
  - Alert system for unrecognized faces (intrusion detection)
  - Interactive alerts page with filter buttons (All/Unreviewed/Reviewed),
    hover zoom, and a full-screen lightbox modal with image download
  - CSV/Excel export of attendance reports
  - Compatibility fix for Python 3.14 (pkg_resources replacement)

The system achieves high recognition accuracy using averaged 128-dimensional
face encodings trained from 50 sample photos per user, and supports multiple
camera feeds. It is designed to be cost-effective, scalable, and easy to
deploy.

Keywords: Face Recognition, Computer Vision, OpenCV, Deep Learning,
          Attendance System, Python, Flask, Real-time Monitoring,
          getUserMedia, Multi-sample Training
